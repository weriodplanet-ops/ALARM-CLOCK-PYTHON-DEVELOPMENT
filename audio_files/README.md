# Audio Files Directory

This directory contains the alarm tone audio files used by the alarm clock application.

## Available Tones

Place your audio files in this directory and they will be automatically detected by the application.

### Default Tone Mappings

The application expects the following audio files:

- **digital.wav** - Sharp digital beep
- **classic.wav** - Classic alarm clock sound
- **gentle.wav** - Soft and gradual wake-up tone
- **buzzer.wav** - Loud buzzer sound
- **chime.wav** - Pleasant chime notification

## Audio File Format

- **Format:** WAV (Waveform Audio Format)
- **Sample Rate:** 44100 Hz recommended
- **Bit Depth:** 16-bit
- **Channels:** Mono or Stereo

## Adding Custom Tones

1. Create or download your alarm tone audio files (WAV format recommended)
2. Place them in this directory with the names listed above
3. Restart the alarm clock application
4. Your custom tones will be available in the Tone dropdown

## Recommended Sources

- Online tone generators (e.g., Audacity)
- Freesound.org (free sound effects)
- Zapsplat.com (royalty-free audio)
- Your own recordings

## Tips

- Keep file sizes reasonable (1-5 MB)
- Test the volume before setting as alarm
- Shorter loops (1-3 seconds) work best for alarms
- Ensure consistent audio levels across files
